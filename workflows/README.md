# AI-Optimized Workflows

This directory contains the complete workflow system for AI-assisted development.

## Structure

- **`prompts/`** - AI instruction files (markdown)
- **`tasks/`** - Executable task scripts following `verb-adjective-noun` naming
- **`tmux/`** - Menu system and configuration

## Usage

Launch the workflow menu with `<prefix> W` in tmux.
